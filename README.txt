== Description ==

WP Google follow Button Plugin is a kind of two in one functinality. It consist of both google plus follow button and google plus badge button. You can use both by shortcode and widget. As there is a single setting page in WP setting in admin panel named Google Plus Follow Setting, so you can customize your setting for shortcode and widget. In widget you need to mention only title. Shortcode for google plus follow button is [viva-google-follow] and short code for google plus badge button is [viva-google-badge-follow].

=You can customize setting for following features:=
* For google plus follow button: 
* You can display follow count in horizontal bubble or vertical bubble or diable show count.
* You can customize button size as small or medium or large.
* You can choose your own language on button.
* You have option to display google plus follow button using shortcode or widget or both.

= For google plus follow button:=

* You can adjust badge width accordingly from backend.
* You can set your badge layout in landscape or portrait from backend.
* You have choise to display cover photo or not in badge.
* Here also you can set your own language in badge.
* You have option to display tagline in badge or not.
* You can adjust theme color badge from backend.
* POT file is added for plugin translation into your languages.
* You have option to display For google plus follow button using shortcode or widget or both.

== Installation ==

1. Download the "Wp Google Follow" Plugin (Plugin zip file).
2. Extract it in the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Issues with google plus profile URL =
* You have write google plus profile URL in admin side in both the options i.e. in google plus follow button and google plus badge button in Profile URL field. Like: "https://plus.google.com/u/0/xxxxxxxxxxxxxxxxxxxxx" here "x" is google plus profile URL number series.
